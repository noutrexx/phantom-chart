# Security Policy

## Supported versions

The main branch is the only supported version.

## Reporting a vulnerability

If you find a security issue, open a private GitHub security advisory when available or contact the maintainer through GitHub.

Please do not open a public issue for exploitable vulnerabilities.

## Scope notes

Phantom Eats is a frontend simulation. It must not process real payments, send card data to a server, or store sensitive payment information.
